# MiFloV2
a KISS implementation of the MiFlo Smart Clock

- run threadMachine.py as root



current setup and connections:

I used BCM pin numbering instead of BOARD pins


Peripherals -> PI

LED matrix

- 5v -> 5v
- GND -> GND
- DIN -> 18

Push button

- Leg1 -> 3.3V
- Leg2 -> 15
       -> 10k resistor -> GND
